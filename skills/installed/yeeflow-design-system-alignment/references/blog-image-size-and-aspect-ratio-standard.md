Yeeflow Blog Image Size and Aspect Ratio Standard

Purpose
Use this standard for Yeeflow blog visual assets so the featured image and in-article images follow different rules appropriate to their roles.

Core principle
- Featured image = fixed, reusable blog hero standard
- In-article images = flexible, content-led editorial support visuals

1. Featured image standard

Use one fixed standard size for all Yeeflow blog featured images.

Default format
- Size: 1200 x 630 px
- Aspect ratio: 1.91:1
- Format: PNG preferred

Use for
- blog cover image
- article hero image
- article preview image
- social/share-friendly blog header image

Purpose
- establish the article theme
- provide a premium brand-forward first impression
- support consistent blog presentation across the site

Visual style guidance
- hero-style
- premium
- modern
- calm
- structured
- enterprise-ready
- slightly more expressive than in-article images, while still following Yeeflow brand standards

2. In-article image standard

Do not force one universal ratio for all in-article visuals.

Use content-led aspect ratios based on the section, the message, and the type of visual needed.

Recommended aspect ratio range
- 3:2 for explanatory visuals
- 4:3 for diagrams, architecture blocks, and card-based visuals
- 5:4 for compact editorial support graphics
- other ratios only when the content clearly justifies them

Purpose
- support readability
- explain section-level ideas
- reinforce the narrative
- make the article more structured and editorial

Visual style guidance
- more editorial
- more structured
- sharper
- more content-led
- less atmospheric than the featured image
- tighter composition
- less decorative empty space

3. Role distinction

Featured image
- article-level visual narrative
- stronger brand presence
- fixed size and ratio
- used consistently across blog posts

In-article images
- section-support visuals
- flexible ratio
- based on the content itself
- should look like part of one editorial system, but not like duplicate hero banners

4. Spacing rule

Featured image
- allow more breathing room and broader composition

In-article images
- keep layouts tighter
- avoid unnecessary empty space
- let the content fill the frame more naturally

5. Yeeflow brand rules

Apply the same Yeeflow brand and design standards to both image types:
- dominant palette:
  - #146FF6
  - #7C4DFF
  - #E85ECF
  - #333333
  - white / soft neutrals
- avoid teal/green drift
- use correct Yeeflow logo rules
- if exact logo fidelity is uncertain, leave a clean logo-safe area instead of generating a substitute mark

6. Recommended prompt wording

Use this when asking for blog visuals:

Featured image:
- create this as a Yeeflow blog featured image
- size: 1200 x 630 px
- aspect ratio: 1.91:1
- use it as the article hero / cover image
- keep it premium, modern, calm, structured, and enterprise-ready

In-article images:
- create these as Yeeflow in-article editorial support visuals
- use content-led aspect ratios based on each section
- do not force a fixed 16:9 or hero-banner ratio
- keep them tighter, sharper, and more explanatory than the featured image
- avoid unnecessary empty space

7. Decision rule

If the image is the article cover:
- use 1200 x 630 px, 1.91:1

If the image is inside the article body:
- choose the ratio based on the content and section needs
- prioritize clarity, editorial fit, and tighter composition
