# LinkedIn featured image standard

Use this reference when the task is to create or review a **Yeeflow LinkedIn featured image** or a **LinkedIn post image series**.

## Default format

- 1080 x 1350 px
- 4:5 ratio
- PNG preferred

Use this for:
- weekly LinkedIn post series
- product update posts
- thought leadership posts
- campaign posts
- image-first corporate-page posts

Do not use this as the default for blog link preview images. For blog or article preview images, use 1200 x 627 px only when the task is specifically a link preview image.

## Layout system

Use a 3-zone vertical layout:

### Zone A — brand / header
- place one main Yeeflow logo at top-left or top-right
- optional small category label
- keep placement consistent across the whole series

### Zone B — main message
- short headline
- optional one-line support text
- keep headline to 2–4 lines maximum

### Zone C — visual support
- product UI frame
- capability cards
- simplified diagram
- structured brand visual tied to the post

## Safe margins

For 1080 x 1350:
- left/right: 72–96 px
- top: 72–96 px
- bottom: 72–96 px

## Logo rules

- light background: use `assets/Yeeflow_logo_Standardx25.png`
- dark background: use `assets/Yeeflow_logo_Whitex25.png`
- one main logo only
- no small extra logos inside mock UI
- if fidelity is uncertain during generation, leave a clean logo-safe area

## Color rules

Use only these as the dominant palette:
- #146FF6
- #7C4DFF
- #E85ECF
- #333333
- white / soft neutrals

Avoid teal and green drift.

## Visual tone

The image should feel:
- modern
- calm
- structured
- enterprise-ready
- premium
- product-led

## Series consistency

Keep these fixed across a series:
- canvas size
- logo placement
- palette balance
- spacing rhythm
- headline hierarchy
- component styling

Let these vary by post:
- headline
- supporting line
- focal visual
- capability cards or diagram
