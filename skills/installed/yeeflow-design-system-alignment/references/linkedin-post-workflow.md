# LinkedIn post workflow

Use this workflow for **Yeeflow corporate-page LinkedIn posts** first. Personal or executive variants are out of scope unless the user explicitly asks.

## Priority

For LinkedIn tasks, apply in this order:
1. `references/master-design-standard.md`
2. the best-fit material module
3. `references/linkedin-featured-image-standard.md` for image format/layout
4. `references/linkedin-post-prompt-templates.md` for reusable prompting patterns

## Creation workflow

1. Read the uploaded post document or pasted copy.
2. Identify all posts in order.
3. Work on **one post at a time** unless the user explicitly asks for batch work.
4. Summarize the current post in 2–3 lines.
5. Propose one featured image direction before generating.
6. Generate the featured image using the LinkedIn featured image standard.
7. Keep the visual system consistent across the series.
8. Wait for approval before moving to the next post unless the user asks to continue.

## Copy workflow (secondary)

Support copy generation or review when asked, but keep it secondary to the visual workflow.

For corporate-page copy:
- keep tone enterprise-ready, product-led, and practical
- align to Yeeflow’s 2026 positioning
- avoid generic AI hype
- avoid overclaiming roadmap maturity
- connect the post to real business value or product direction

When reviewing copy, check:
- clarity of the main message
- tone fit for a Yeeflow corporate page
- alignment to actual roadmap maturity
- whether the CTA is concise and appropriate

## Image-generation rules

- use the LinkedIn featured image standard by default
- treat the image as a custom LinkedIn post visual, not a blog hero image
- if exact logo fidelity cannot be guaranteed, leave a clean logo-safe area instead of generating a substitute logo
- keep the image suitable for a multi-post weekly series
