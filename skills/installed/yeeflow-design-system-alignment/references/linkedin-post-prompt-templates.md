# LinkedIn post prompt templates

Use these templates when the user uploads or pastes multiple LinkedIn posts and wants to work through them one by one.

## Template 1 — start with post 1

Use when the user uploads a document containing multiple LinkedIn posts and wants to start with the first one.

```text
I uploaded a Word document that contains 5 Yeeflow LinkedIn posts for this week.

Please do the following:
1. Read the document and identify all 5 posts in order.
2. For now, work only on Post 1.
3. Summarize Post 1 in 2–3 lines.
4. Propose one LinkedIn featured image concept for Post 1.
5. Then generate the featured image for Post 1.

Important requirements:
- Use the project instruction, the installed yeeflow-design-system-alignment skill, and the LinkedIn image standard file.
- This must be a Yeeflow LinkedIn featured image, not a blog hero image.
- Use the Yeeflow LinkedIn featured image standard:
  - 1080 x 1350 px
  - 4:5 ratio
  - suitable for image-first LinkedIn posting
- Keep the image visually consistent with a 5-post weekly series.
- Use Yeeflow’s approved visual direction:
  - modern
  - calm
  - structured
  - enterprise-ready
  - premium
  - product-led
- Use only the dominant Yeeflow palette:
  - #146FF6
  - #7C4DFF
  - #E85ECF
  - #333333
  - white / soft neutrals
- Avoid teal/green drift.
- Use a 3-zone vertical layout:
  - Zone A: brand/header
  - Zone B: main message
  - Zone C: visual support
- Keep the logo placement consistent with future posts in the series.
- Use the correct Yeeflow logo rules.
- If exact logo fidelity cannot be guaranteed, leave a clean logo-safe area instead of generating a substitute logo.
- Do not work on Posts 2 to 5 yet.

Before generating the image, first show me:
- the Post 1 summary
- the proposed image direction
- the headline text to place on the image
- the supporting line, if any
```

## Template 2 — generate after concept approval

```text
Proceed to generate the featured image for Post 1 using the approved concept.

Keep it fully aligned with:
- the project instruction
- the installed yeeflow-design-system-alignment skill
- the LinkedIn featured image standard

Remember:
- 1080 x 1350 px
- 4:5 ratio
- LinkedIn series consistency
- Yeeflow standard palette only
- clean logo-safe area if exact logo fidelity is uncertain
```

## Template 3 — move to the next post

```text
Now work on Post [number] from the uploaded LinkedIn post document.

Please:
1. summarize the post
2. propose the featured image direction
3. generate one LinkedIn featured image

Use the same Yeeflow LinkedIn featured image system as the previous posts so the full series stays visually consistent.
```

## Template 4 — copy review or generation (secondary)

```text
Review this Yeeflow LinkedIn corporate-page post copy.

Check:
- enterprise-ready tone
- product-led messaging
- alignment with Yeeflow’s 2026 positioning
- roadmap realism
- CTA clarity

Then provide:
1. what works
2. what is too generic or hype-heavy
3. a revised version
```
