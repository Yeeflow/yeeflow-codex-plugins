# Docs

This folder stores sanitized Yeeflow learning notes, feature studies, validation summaries, and runtime import evidence.

Good candidates for this folder:

- feature pattern studies
- generated baseline summaries
- validator behavior notes
- import/runtime test summaries
- rebuild plans and staged learning specs when they do not contain secrets

Avoid storing raw decoded exports or full customer package payloads here. If detailed structure is needed, summarize the pattern and redact tenant-specific or sensitive values.
